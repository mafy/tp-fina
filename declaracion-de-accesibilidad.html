Cronos - Declaración de Accesibilidad

Desde Cronos nos comprometemos a proporcionar una plataforma web que sea accesible para la mayor audiencia posible,
independientemente de las circunstancias y sus capacidades. Para cumplir este compromiso, evaluamos y verificamos periódicamente nuestro sitio.

Además, nuestro objetivo es alinearnos lo más posible a las Pautas de Accesibilidad del Contenido en la Web (WCAG), publicadas por el Consorcio
World Wide Web (W3C) para el cual alcanzamos un nivel de conformidad AA. Estas pautas explican cómo hacer el contenido más accesible para las
personas con discapacidad. El cumplimiento de las mismas nos ayudará a proporcionar un sitio que sea más fácil de usar para todos.

Cronos hace todo lo posible para proporcionar la mejor experiencia a todos los visitantes de nuestro sitio. Por ejemplo, incluimos:
Transcripciones de las clases para permitir que los visitantes con discapacidad auditiva puedan consumir el contenido.

Texto alternativo de imágenes para permitir a los usuarios el uso de lectores de pantalla y aquellos con discapacidad visual puedan identificar y
dar sentido a imágenes útiles.

Formación de etiquetas y controles que estén asociados con la solicitud correspondiente.
Contenido que se puede ampliar con los controles estándar del navegador.
Artículos CAPTCHA auditivos y visuales para proporcionar formas alternativas de consumir esta información.

La versión actualmente visible de este sitio es de Noviembre del 2021, fecha en la que se hizo la última revisión del nivel de accesibilidad.

Si tiene algún comentario o sugerencia para mejorar la accesibilidad de nuestro sitio, no dude en comunicarse con nosotros enviando un correo
electrónico a accesibilidadcronos@cronos.com. Sus comentarios nos ayudarán a hacer mejoras.
